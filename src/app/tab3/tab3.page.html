ASL Detector Detector de Lenguaje de Señas

Esta aplicación utiliza la IA de Teachable Machine para analizar imágenes correspondientes al Lenguaje de Señas Americano (ASL) y posteriormente indicar a qué letra se está refieriendo, para que las personas con problemas auditivos se puedan comunicar fácilmente con cualquier otra persona.
